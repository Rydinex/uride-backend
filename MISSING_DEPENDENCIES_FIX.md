# ✅ **QUICK FIX - Missing Dependencies**

## 🔧 The Problem

```
Error: Cannot find module 'axios'
```

This means npm packages aren't installed yet.

---

## ✅ The Fix (Copy & Paste)

You're in the backend folder already. Just type:

```powershell
npm install
```

Press Enter and wait (30-60 seconds)

---

## After Installation Completes

Then type:

```powershell
npm start
```

Press Enter

You'll see:
```
MongoDB connected
Server running on port 4000
```

✅ **Done!**
